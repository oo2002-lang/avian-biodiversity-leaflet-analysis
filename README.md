# Avian Biodiversity & Anthropogenic Landscape Analysis
Analysis of avian biodiversity across ~85 field sites in the Hudson Valley, examining the impact of anthropogenic landscapes on bird species distribution. Data not included due to confidentiality

# # Methods
Field surveys conducted as part of Bard Summer Research Institute (BSRI) 2025

Shannon Diversity Index mapping in R

Spatial visualization using Leaflet

Tools & Languages:

R / Quarto (.qmd)

Leaflet

tidyverse
